ヘルプ

ダッシュボード

チェックの実行

「全てチェック」ボタンで全分院の空き状況を一括チェックします。システム別（dent-sys / Stransa / GMO）に個別チェックも可能です。

チェック中は経過時間が表示されます。完了まで通常2〜3分かかります。

フィルタ

システムフィルタ: All / dent-sys / Stransa / GMO で表示を絞り込めます。

職種フィルタ: All / Dr（歯科医師）/ DH（歯科衛生士）で絞り込めます。

結果の見方

OK（緑）: 空き枠が閾値以上あり、新規予約を受け入れ可能な状態です。

NG（赤）: 空き枠が閾値未満で、新規予約の受け入れが難しい状態です。

タイムラインバーでは、色付きの部分が空き枠を示します。赤い縦線は現在時刻です。

詳細表示

各クリニックカードの「詳細を見る」をクリックすると、スタッフごとの空き枠時間帯が確認できます。

スタッフ管理

スタッフ同期

「スタッフ同期」ボタンで、各予約システムから最新のスタッフ一覧を取得します。新しいスタッフが追加された場合はここで同期してください。

有効/無効の切替

各スタッフ左のトグルスイッチで、チェック対象に含めるかどうかを設定します。無効にしたスタッフの空き枠はカウントされません。

「自動除外」と表示されているスタッフは、予約システム上で見つからなかったため自動的に無効化されています。

職種の設定

Dr（歯科医師）/ DH（歯科衛生士）/ 矯正（矯正歯科医）のボタンで職種を分類できます。職種ごとに閾値を設定したり、フィルタで絞り込んだりできます。

WEB予約受付

「WEB」タグをONにすると、そのスタッフがWEB予約を受け付けていることを示します。表示用のラベルです。

初診枠の閾値

各クリニックの上部で、職種別の初診枠閾値（分）を設定できます。空き枠がこの閾値未満の場合、NGと判定されます。

例: Dr閾値=30分 → 30分以上の空きがあればOK、なければNG

メモ・タグ

メモ: スタッフごとに自由にメモを残せます（例: 「午前のみ」「産休中」）。

タグ: 「+ タグ」ボタンで任意のタグを追加できます。分類や目印として活用してください。

分院管理

分院の有効/無効

トグルスイッチで各分院のチェック対象を切り替えます。

無効にした分院は、チェック実行時にスキップされます。一時的にチェック不要な分院がある場合に使用してください。

表示情報

システム: dent-sys または Stransa の予約システム種別を表示しています。

ID: 予約システム上の分院識別子です。

ルール設定

連続スロット数

空き枠1ブロックとしてカウントする連続スロット数です。デフォルトは6（= 5分 x 6 = 30分）。30分以上の連続した空きを1ブロックとして数えます。

必要空き枠数

OKと判定するために必要な最低ブロック数です。この値以上の空きブロックがあればOK、未満ならNGとなります。

除外パターン

チェック対象から除外するキーワードをカンマ区切りで入力します。スタッフ名にこのキーワードが含まれる場合、チェック対象外になります。

チェック時間帯

空き枠を確認する時間帯を設定します（例: 9時〜19時）。この範囲外の時間帯は無視されます。

スロット間隔

予約システムの最小時間単位（分）です。通常は5分です。

結果一覧

過去の結果を閲覧

日付セレクトボックスから過去のチェック結果を選択して閲覧できます。

フィルタ

ダッシュボードと同様に、システム別・職種別のフィルタが使えます。

充足/不足の表示

充足（緑）: 空き枠が閾値以上あり問題なしの状態です。

不足（赤）: 空き枠が閾値未満で対応が必要な状態です。

各カードを展開すると、スタッフ別の空き枠詳細とタイムラインが確認できます。

月次レポート

概要

月ごとに各分院の空き枠状況を集計したレポートです。どの分院が頻繁に空いているか、空き枠の多さを一覧で把握できます。

対象月の選択

上部の月選択で表示したい月を選び、「表示」ボタンを押します。過去の月も選択可能です。

テーブルの見方

空き日数 / 診療日: その月でチェック閾値以上の空きがあった日数 / 分院ごとの診療日数。診療日数は休診曜日・祝日を除外して算出しています。

頻度: 空き日数 / 診療日数の割合をバーとパーセントで表示。100%に近いほど毎日空きがある状態です。

合計ブロック: 月間の空き枠ブロック合計数。1ブロック = 30分の空き枠です。

日平均: 1日あたりの平均ブロック数。日常的にどのくらい空いているかの目安です。

Dr / DH: 歯科医師（Dr）と歯科衛生士（DH）それぞれの月間ブロック合計。どちらの職種に空きが偏っているか確認できます。

ソート

開院順: デフォルトの分院並び順。

頻度順: 空き枠が出やすい分院が上に来ます。空き枠対策の優先度確認に便利です。

ブロック数順: 合計ブロック数が多い順。空き枠の総量で比較できます。

CSV出力

「CSV出力」ボタンで、表示中のデータをCSVファイルとしてダウンロードできます。Excelで開いて分析や報告書作成に活用できます。
